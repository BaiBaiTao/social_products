⚠ Hard Rules

Must always be followed — violation of any rule invalidates the extraction.

Rule 1 — Explicit Product Name
Product name MUST appear explicitly in the content text. Do NOT guess or complete names from knowledge.
Exception — Collaborations: If text describes a collab (e.g. "X has reunited with Y to put his spin on Z"), construct name as "CollabPartner x ProductLine"
✓ "@lorenz.og … his latest spin on the Air Max 95" → name = "Lorenzo x Air Max 95"
Co-Brand Preservation: If text ties two brands to the SAME product, retain both entities.
✓ "Crocs and LEGO … Brick Clog" → name = "LEGO x Crocs Brick Clog"
✗ Do NOT drop, replace, or normalize away an explicitly mentioned collaborator token.
Rule 2 — Brand Identification
Brand MUST be identifiable from text (via @mention, explicit name, or hashtag). Use the OFFICIAL brand name.
✗ Retailers (Ulta, Sephora, Amazon, Target) are NOT brands.
Rule 3 — Evidence
For each product, provide evidence: copy the EXACT substring from the original text containing the product name.
Rule 4 — Relevance Judgment
Judge each product's relevance for homepage display:
on_topic Real, purchasable consumer product suitable for shopping homepage
off_topic Falls under Global Exclusions, promotional freebies, subscription services (always off_topic)
borderline Unusual but still a legitimate purchasable item
Only extract on_topic or borderline. Do NOT extract off_topic products.
Rule 5 — Product Name Completeness
Single-word product names must be appended with product type/category for searchability.
✗ WRONG: name="Rapunzel", category="fragrance" → searching "Disney Rapunzel" finds a movie character
✓ RIGHT: name="Rapunzel Fragrance", category="fragrance" → finds the product
Multi-word names like "Soft Pinch Liquid Blush" are already specific — do NOT add category.
Rule 6 — Language Check (MUST be done FIRST)
Determine the PRIMARY language of the post. Ignore brand names, product model numbers, and hashtags.
If NOT English → immediately return empty products with failType="non_english".
Do NOT extract products even if brand/product name appears to be English.

✓ Homepage Inclusion Criteria

ALL three criteria must be satisfied. If any fails, do NOT include in products.

1) Purchasable
The item is a real consumer product/service that can be obtained independently.
2) Searchable
The text provides enough identity to search it (at least brand + product name/type).
3) Primary Focus
The product is a main subject of the post (review/recommendation/demo), not incidental tags.
Campaign/engagement slogans without concrete product evidence (e.g. #JoinTeamX, #XChallenge, #Giveaway) → cap at confidence 2.

★ Confidence Scoring System

List ALL product-like mentions as candidates. The confidence score (0-9) determines how specific the product identification is — replacing any binary yes/no extraction rule.

Scoring Principles:
  • Score based on how clearly the text identifies a product — NOT on whether you recognize the product.
  • You cannot search the web. Do NOT penalize unfamiliarity.
  • Trust the text's signals: hashtag as name, brand handle + product term, "must have" language.
  • If uncertain between adjacent scores, choose the LOWER score.

📊 Score Definitions (0-9)

9
Full Variant — Specific name + colorway/edition/variant + brand
e.g. "Air Jordan 1 High Alaska", "Soft Pinch Liquid Blush in Joy"
8
Clearly Specific & Searchable — Brand + distinct product name, missing variant detail
Brand + distinct product name that reads near-unique in context; specific collab/product name directly searchable
7
Specific with Minor Ambiguity — Model/version/collab/distinct name
  • Model/version numbers: "EP-133 sampler", "991v1", "990v4"
  • Collab version: "Lorenzo x Air Max 95", "Travis Scott x Air Jordan 1"
  • Co-branded: "LEGO x Crocs Brick Clog" (both entities explicit)
  • Hashtag as product name: "#AutoScooper" (not campaign slogans)
  • Single-product review/recommendation post with identifiable name + brand
6
Product Line/Series — Named range without version; narrow type with endorsement
  • Product series: "Air Max 95", "Air Max 90", "KERATIME" (series numbers 95/90/97 are NOT model numbers)
  • Collab product without collab name captured → stays at 6
  • Brand + narrow product type as main subject with strong endorsement
✓ Qualifies for 6: "cushion foundation from @skintific" (narrow type, specific review)
✗ Does NOT qualify: "Nike sneakers" (too broad) → stays at 5
── ✂ Extraction Threshold (≥ 5) ──
5
Brand + Single Category — Broad but usable product type
e.g. "Nike sneakers" (single category), "Adidas hoodie" (single category)
4
Mixed Collection — Brand + multi-category bucket
e.g. "Nike new arrival", "Adidas spring collection", "merch", "capsule", "drop", "kit"
3
Very Weak — Brand/category hints only, low searchability
2
Weak Signal — Explicit product-like signal but not concretely searchable
e.g. campaign/engagement slogans, weak brand+term combinations
1
Uncertain — Seems product-like but confirming requires guessing
0
Not a Product — Person's name, movie title, location, etc.

📋 Key Scoring Rules

  • → 7+ Model/version numbers (e.g. "2010", "v1", "v2", "133")
  • → 7+ Collaboration name included in product name (e.g. "Lorenzo x Air Max 95")
  • → 7+ Hashtag clearly denoting a product name (e.g. "#AutoScooper")
  • → cap 2 Campaign/engagement slogans (#JoinTeamX, #XChallenge, #Giveaway) with no product evidence
  • → 6 Product series/line (e.g. "Air Max 95", "KERATIME") — series numbers are NOT model numbers
  • → 6 Collab product WITHOUT the collab name captured → missing specificity
  • → 6 Brand + narrow product type as main subject with strong endorsement
  • → 5 Brand + single broad product type (tee, sneakers, hoodie)
  • → 5 Brand + product type without strong endorsement
  • → 4 Mixed collection bucket (new arrival, spring collection, trending picks, kit)
  • → 4 Generic collection words: "merch", "capsule", "drop", "essentials" (unless narrowed to single type → 5)
  • → 4 Broad bucket words: "gear", "apparel", "products", "cosmetics", "makeup", "skincare" (unless narrowed → 5)
  • → ≤ 4 Brand-only or handle-only mention with no concrete item type (never 5)
  • → 3-4 Pure category word without enough product identity
  • → cap 5 Implicit promotion (@brand product without endorsement language) → cap at 5
Tie-breaker: If uncertain between adjacent scores, always choose the LOWER score.

🛡 Spam / SEO Tag Detection

Must be checked BEFORE finalizing scores.

Compare post TITLE to products found.
If the title indicates a SPECIFIC TOPIC (game, challenge, movie reaction, comedy skit) and product mentions appear ONLY in description hashtags or keyword-stuffed text UNRELATED to that topic → those are SEO spam tags.
Action: Cap ALL spam-tag products at confidence 2, regardless of how specific the model name looks.
Example:
Title: "Rescuing My Roblox Friends from Golden Tsunami"
Description: "a01,a11,a10,a20,a30,…,iphone, #Iphone8plus, #j5prime, #j2prime"
→ Post is about Roblox gameplay; phone models are spam SEO tags → score 1-2 for all.

Only apply when mismatch is CLEAR. If title is vague or products plausibly relate, score normally.

⚡ Critical Rule

After scoring all candidates, populate "products" with EVERY candidate whose confidence ≥ 5 that is "on_topic" or "borderline" AND satisfies all homepage inclusion criteria.

This is a hard rule — do NOT override based on subjective judgment.
"products"
Only candidates with confidence ≥ 5
+ relevance (on_topic / borderline)
+ homepage criteria passed
"candidates"
